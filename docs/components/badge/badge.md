## API

### Attributes

| Name | Default | Options | Notes |
|---|---|---|---|
| `color` | `primary` | `danger`, `primary`, `secondary`, `success`, `warning` | - |
| `size` | `md` | `sm`, `md`, `lg` | - |

### Slots

| Name | Notes |
|---|---|
| Main | Contains the text content of the badge |
